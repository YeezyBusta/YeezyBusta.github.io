ATM

What is ATM?

ATM stands for applied mathematics magnet. It is a 3 year class at Bancroft middle school where we learn how to use technology to it's full capacity.

6th Grade

In 6th grade ATM, we learned how to code using Javascript. We created a picture using what we learned in Javascript. We also started typing everyday.

7th Grade

In 7th grade, we learned how to use 3 major applications. We used Photoshop, Illustrator, and Sketchup. We also learned how to use Garageband.

8th Grade

So far, in 8th Grade we have learned how to code at a much higher level than before. We have coded small websites like the one we did in dash. Now we are learning how to create and edit a video.
